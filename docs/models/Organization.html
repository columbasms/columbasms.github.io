Organization

  • id : Integer
    L'id dell'organizzazione.
  • organization_name : String
    Il nome dell'organizzazione.
  • avatar_normal : String
    Il link dell'immagine di profilo dell'organizzazione
  • cover_normal : String
    Il link dell'immagine di copertina dell'organizzazione
  • description : String
    Una breve descrizione dell'organizzazione, dei suoi scopi e delle sue iniziative
