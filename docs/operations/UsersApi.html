This is the API

usersPost

Questo endpoint regitra l'utente alla piattaforma

URL

https://www.columbasms.com/api/v1/users

HTTP Method

POST

Response Type

DigitsClient

Parameters

  • Header: X-Auth-Service-Provider String

    Il provider per performate l'autenticazione

  • Header: X-Verify-Credential-Authorization String

    Le credenziali per l'autenticazione

  • Header: gcm-token String

    Il token di registrazione per Google Cloud Messaging

usersIdGet

Ritorna il singolo utente

URL

https://www.columbasms.com/api/v1/users/{id}

HTTP Method

GET

Response Type

DigitsClient

Parameters

  • Path: id Integer

    L'id dell'utente

GET

List
