This is the API

topicsGet

Ritorna tutti i topic

URL

https://www.columbasms.com/api/v1/topics

HTTP Method

GET

Response Type

List

Parameters

  • Query: order_field String

    Il campo di ordinamento delli topic

  • Query: order_type String

    Il tipo di ordinamento che bisogna attuare. DEVE essere o *asc* o *desc*

  • Query: limit Integer

    Il numero massimo di elementi che devono essere ritornati.

  • Query: offset Integer

    Da quale elemento nel database deve partire l'array
