This is the API

campaignsGet

L'endpoint *campaigns* ritorna la lista di tutte le campagne disponibili,\nsia scadute che non.

URL

https://www.columbasms.com/api/v1/campaigns

HTTP Method

GET

Response Type

List

Parameters

  • Query: order_field String

    Il campo di ordinamento delle campagne

  • Query: order_type String

    Il tipo di ordinamento che bisogna attuare. DEVE essere o *asc* o *desc*

  • Query: limit Integer

    Il numero massimo di elementi che devono essere ritornati.

  • Query: offset Integer

    Da quale elemento nel database deve partire l'array
